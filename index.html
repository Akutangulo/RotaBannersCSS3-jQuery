Bienvenido a las páginas de GitHub.

Este generador automático de la página es la manera más fácil de crear páginas bonitas para todos sus proyectos. El contenido Autor página aquí usando Markdown sabor GitHub, seleccione una plantilla hecha a mano por un diseñador, y publicar. Después de que su página se genera, se puede ver la nueva rama branch:

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

Si usted está utilizando el GitHub para Mac, simplemente sincronizar el repositorio y verás la nueva rama.

Plantillas de diseño

Hemos elaborado algunas plantillas hermosas para su uso. Seguir adelante y continuar diseños para navegar a través de ellos. Usted puede volver a editar su página antes de publicar. Después de publicar su página, usted puede visitar el generador de página y cambiar de tema. El contenido de la página se conservarán si queda formato Markdown.

En lugar Drive Stick?

Si prefiere no utilizar el generador automático, empujar una rama llamada gh-pages para su repositorio para crear una página manualmente. Además de apoyar el contenido HTML normal, las Páginas de GitHub apoyan a Jekyll, un sencillo, generador de blog estático escrito por Tom Preston-Werner. Jekyll hace que sea fácil de crear todo el sitio encabezados y pies de página sin tener que copiarlos a través de cada página. También ofrece soporte inteligente para blogs y otras funciones avanzadas y plantillas.
